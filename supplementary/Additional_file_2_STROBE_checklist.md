# Additional file 2. STROBE Reporting Checklist

STROBE checklist for cross-sectional studies, completed for the manuscript *"A Multiple Correspondence Analysis Approach to Mitigate the Non-Communicable Disease (NCD) Epidemic: Introducing a Health-Related Behavior-Based NCD Vulnerability Index."* Section names and appendix letters refer to the main manuscript.

| STROBE item | Recommendation | Location in manuscript |
|---|---|---|
| 1 | Title and abstract: study design and balanced summary | Title page; Abstract |
| 2 | Background and rationale | Introduction |
| 3 | Objectives | Introduction; Abstract |
| 4 | Study design | Data and Variables (Source of Data); Methodology |
| 5 | Setting | Data and Variables (Source of Data); Appendix D |
| 6 | Participants: eligibility and selection | Data and Variables (Source of Data); Appendix D |
| 7 | Variables: outcome, exposures, confounders | Data and Variables (Variables Used in Analysis) |
| 8 | Data sources and measurement | Data and Variables; Methodology |
| 9 | Bias | Methodology (endogeneity sensitivity index); Discussion |
| 10 | Study size | Data and Variables (N = 12,650); Appendix D |
| 11 | Quantitative variables and groupings | Data and Variables (tiers, age bands); Methodology |
| 12 | Statistical methods | Methodology (MCA; multilevel model; validation) |
| 13 | Participants: numbers at each stage | Source of Data (participant flow); Appendix D |
| 14 | Descriptive data | Results (Bivariate Analysis); Appendix D |
| 15 | Outcome data | Results (Bivariate Analysis) |
| 16 | Main results: adjusted estimates with confidence intervals | Results (Multilevel Logistic Regression Analysis) |
| 17 | Other analyses: subgroups and sensitivity | Results (Sensitivity NCDVI; MCA Robustness; External Validation; Decision Curve Analysis) |
| 18 | Key results | Discussion |
| 19 | Limitations | Discussion |
| 20 | Interpretation | Discussion |
| 21 | Generalisability | Discussion (External Validation on BDHS 2022) |
| 22 | Funding | Declarations |

Reference: von Elm E, Altman DG, Egger M, Pocock SJ, Gotzsche PC, Vandenbroucke JP; STROBE Initiative. The Strengthening the Reporting of Observational Studies in Epidemiology (STROBE) statement: guidelines for reporting observational studies. Lancet. 2007;370(9596):1453-1457.
