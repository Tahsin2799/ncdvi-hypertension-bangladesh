# Additional file 1. TRIPOD Reporting Checklist

Completed TRIPOD checklist (Transparent Reporting of a multivariable prediction model for Individual Prognosis Or Diagnosis) for the manuscript *"A Multiple Correspondence Analysis Approach to Mitigate the Non-Communicable Disease (NCD) Epidemic: Introducing a Health-Related Behavior-Based NCD Vulnerability Index."* Section names and appendix letters refer to the main manuscript.

| 7a, 7b | Predictors | Data and Variables (ten NCDVI inputs; confounders) |
| 8 | Sample size | Data and Variables (N = 12,650); Appendix D |
